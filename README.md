# NAS Logistics - Marketing Management App

## Omnichannel Customer Engagement System

A comprehensive platform for managing customer communications across email, SMS, and push notifications for logistics services.

### Features
- 🎯 Customer Segmentation & CRM
- 📧 Multi-channel Campaigns (Email, SMS, Push)
- 📦 Automated Shipment Notifications
- 📊 Campaign Analytics & Performance Tracking
- ⚙️ Workflow Automation
- 🔐 Role-based Access Control
- 📱 Responsive Dashboard

### Tech Stack
- **Backend:** Node.js, Express.js
- **Database:** PostgreSQL
- **Frontend:** React.js
- **Message Queue:** Redis
- **Notifications:** Twilio (SMS), SendGrid (Email), Firebase (Push)

### Project Structure
```
NAS-logistics/
├── backend/              # Node.js/Express API
├── frontend/             # React dashboard
├── database/             # PostgreSQL schemas
├── services/             # Notification services
└── docs/                 # Documentation
```
